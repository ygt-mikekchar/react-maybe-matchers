React Maybe Matchers for Jasmine

Tests using the React test utilities directly are both difficult to read and inconvenient to type. These matchers are intended to to make it easier to write tests and to understand what they are doing later. Please see the project page or browse the annotated source code for more information of how to use the matchers.
